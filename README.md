# HyticAdminGUI

HyticAdminGUI is a complete rewrite of the existing AdminGUI plugin, designed to provide server administrators with an intuitive graphical interface for server management.

This rewrite aims to modernize the plugin with:
*   Rewritten from Java to pure Kotlin
*   Targeting newer Minecraft API versions
*   Support for Paper server software
*   Refactored for improved readability and maintainability

This project is based on and inspired by the original AdminGUI developed by zigazajc007.
Original Project: https://github.com/zigazajc007/AdminGUI
